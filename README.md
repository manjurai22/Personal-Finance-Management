FinTrack is a **Django-based personal finance management web app** that helps users track **income, expenses, budgets, debts, and savings goals** in one place.

---

## Tech Stack

- **Backend:** Django (Python)
- **Database:** SQLite (default, can be changed to PostgreSQL/MySQL)
- **Frontend:** Django Templates, HTML, CSS, JavaScript
- **Auth System:** Django built-in authentication

---

## Core Features

- User authentication (login/logout)
- Category-based **income & expense tracking**
- **Monthly budget** monitoring
- **Debt management** (borrowed & lent)
- **Savings/financial goals** with progress tracking
- Interactive **dashboard analytics & charts**
- User-specific secure data handling

---

## How It Works (Brief Logic)

- **Transactions** record income and expenses under categories.
- **Budgets** define monthly spending limits and compare with expenses.
- **Debts** track money owed or lent and affect real balance.
- **Goals** reserve money from balance and show completion progress.
- The **Dashboard** aggregates all data to calculate:
  - Total income, expenses, and savings
  - Today's transaction count
  - Monthly budget usage
  - Debt impact
  - Goal progress
  - Real net balance
